# aegis-mind
AegisMind is a privacy-first, offline-capable AI system built on local language models. It orchestrates multiple specialized agents and optional web retrieval to deliver high-quality responses without sharing user data with proprietary model providers.

## Multi-agente colaborativo
O modo colaborativo executa múltiplos agentes (pesquisador, analista e cético) e sintetiza as respostas em uma única saída. É possível ajustar os papéis em `config.py` via `AGENT_ROLES` e ativar/desativar o modo com `ENABLE_MULTI_AGENT`.

No chat, digite `agentes` para visualizar os papéis ativos.
